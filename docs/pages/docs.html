How it works

  1. Export your model to ONNX (from PyTorch / sklearn / etc.).
  2. Place it at docs/assets/model/model.onnx.
  3. Update metadata.json with input names, shapes, and labels.
  4. Push to main — GitHub Actions deploys to gh-pages.

Replacing the input form

Edit docs/pages/demo.html and docs/assets/js/app.js.
